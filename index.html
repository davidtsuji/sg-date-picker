sgDatePicker

A JavaScript control that turns an input into a jQuery $.datePicker .. all without writing any JavaScript.

View on GitHub »

Example click or tab into the input field to show the date picker

Default Markup the date format defaults to dd/mm/yyyy

<input type="text" data-sg-date-picker="">

Custom Markup with a custom date format

<input type="text" data-sg-date-picker='{ "dateFormat":"yy/mm/dd" }'>
